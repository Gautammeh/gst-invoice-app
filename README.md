# GST Invoice Web App

A print-ready, mobile-accessible GST invoice web application built using pure HTML, CSS, and JavaScript.

## Features
- A4 print-perfect invoice layout
- Mobile-friendly invoice filling
- Auto amount in words (Indian format)
- Save multiple invoices (localStorage)
- Previous saved bills panel
- Safe print without layout break

## Tech Stack
- HTML
- CSS
- JavaScript (No framework)

## Usage
1. Fill invoice details
2. Save invoice
3. Print or reload saved invoices

## Note
This app is frontend-only and works offline using browser storage.
